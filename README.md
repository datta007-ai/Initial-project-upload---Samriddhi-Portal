## Problem
Citizens struggle to discover welfare schemes, track grievances, and access emergency services due to fragmented portals, language barriers, and low digital literacy.

## Solution
SAMRIDDHI is a unified, AI-enabled citizen welfare portal that provides:
- Scheme eligibility discovery
- Grievance filing & tracking
- Health & emergency services
- Multilingual AI assistant
- Glass-morphism accessible UI

## Features
- Client-side intelligent scheme matching
- AI Chat + Voice assistant (Web Speech API)
- Dark / Light accessible theme
- Health & wellness services module
- Grievance redressal workflow (demo)
- SOS & helpline directory

## Tech Stack
- HTML5
- CSS3 (Glassmorphism + AA accessibility)
- Vanilla JavaScript
- Web Speech API
- GitHub Pages compatible

## How to Run
Open `index.html` in any modern browser (Chrome recommended).

## Note
This is a hackathon demo. No real government data is processed.
